Some ROOMS assistance

Q: Can a "Room" be outside?

A: YES. A room is just the name used to describe all locations. There is a Room type field which allows you to be specific about the location type but it is not a required field

Q: What do I do if I do not know the answers to all the fields?

A: Just complete the required fields (with a green border) and then come back to edit the room when you do know the answer. The game will work if just the required fileds are completed, but the game will be very dull

Q: Some bits I need to complete are not visibile here?

A: The form is designed so only parts required are visible. If the field you want is not visible ensure the relevant, previous fields are set correctly. Eg. The "Can be dark" slider needs to be on for the "Dark description" field to be editable

Q: How do I create a line break in the text?

A: You need to enter " br ", including the space before and after the "br".

Q: How do I attach a room image?

A: Once you have created a picture sav e it in the folder "imgs" in your game folder. Then pick it from the Image picker field

Q: Do I have to fill all these fields in?

A: NO. You only need to coplete the required fields (indicated with a green border). However, the more you complete, either now or at a later time, the better the game experience.

Q: How do I insert a new line in the text?

A: You need to type " br " into the text, including the space bfore and after the "br".

Q: Some bits I need to complete are not visibile here?

A: The form is designed so only parts required are visible. If the field you want is not visible ensure the relevant, previous fields are set correctly. Eg. The "Hidden" slider needs to be on for the "Hidden in", "Hidden under" and "Hidden behind" fields to be editable

Q: How do I set what happens if the player "uses" this object?

A: If you want the object to have a specific action when "used", set the "Use action" drop down box to the setting you want. This basically replaces the "use" verb with the selected action.

Q: The value field will not let me enter a "Â£" or "$" symbol.

A: The value field is for numbers only. The money symbol is set at the "Cash Symbol" field in "Settings>Initial settings".

Some EVENTS assistance

Q: What are events?

A: These are routines that occur at random (although some control can be made using the Occurence field) during the game.

Q: Why?

A: They are intended to make locations feel different at different times. They can also be set to affect the mood of the players character

Q: Where do they occur?

A: They can be set to occur anywhere or in one particular location. Up to you.

Q: How are they initiated

A: An internal rolling of dice used with the Occurence field, determines if the event is triggered

Spells
Some SPELLS assistance

Q: What does "Type" mean?

A: Spells have diffrent core types. Currently, these include:

  • Attack - the spell will act like a weapon on an NPC
  • Creation - this will make an object or person available in the game
  • Destruction - this will destroy an object or, if possible, a person. Possibly replacing the object with another
  • Transformation - this changes an object either making the object have a different description or replacing it with an object which is different version of the current one
  • Transportation - this moves either an object, an NPC or the player, to a different location. To move the player, ensure the "Transport self" switch is set to on.

Q: How does the game know where or whom the spell is aimed at?

A: The fields "At person" and "At object" define the respective targets depending upon what these are set to. The field "In one room" should be set to a value if you want the spell to only be useable in one room.

Q: How do I define what object is created or destroyed?

A: The field's "Creates object" and "Destroys object" should be set to values relative to what you need the spell to do. Leave them at "No object selected" if you do not want an object destroyed or created

Q: How do I define which NPC is created or destroyed?

A: The field "Creates NPC" should be set to the NPC name you would like made available to the game after the spell is cast. If you want to destroy an NPC, then set the "Kills NPC" switch to on and select the NPC in the "At person" field. This will mean that when the spell is cast on the required NPC, if successful it will kill them.

Q: Why does the spell need a power?

A: When the spell is cast, it may be that the NPC it is cast on has a stat which could defend against it, or an object maybe strong enough to resist. The power is the spell's stat against the defensive stats.

Q: What is the difference between "Description", "Use text" and "Second use text"?

A: The "Description" is the text displayed when the player examines the spell. The "Use text" and "Second use text" are the text displayed when the player casts the spell for the first and second time respectively.

Until the next version, please follow the below to set up your environment for the creator.

  1. Find the folder called "blankGame" inside the folder "new_Games" in the "VC" folder
  2. Copy the entire folder to a location of your choice (but one you can locate quickly)
  3. Re-name the "blankGame" folder to the name of your game
  4. Copy any images you have for your game into the folder called "imgs" in your new game folder (nb. during the game creation you can select these images and the game will point to them)
  5. A story is a great guideline for everything else you do
  6. DRAW A MAP! it makes things so much easier
  7. Add objects after rooms (which, of course, can be outside locations). You can put objects in rooms.
  8. Add non playable characters before adding conversations
  9. When you have added everything else, add your winning requirements

Hopefully you can see how by just entering these things, you can make the game do an awful lot. Enjoy

Select a room to edit or add a new room, using the menu on the left.

Some important things to note here;

  • The "Room description" is only displayed once and in future is superceeded by "Short description" - this gives you the chance to add extra details and story in the "Room description"
  • To create a forced new line add " br ". Include the spaces either side of 'br'
  • Once you have selected a room to edit, the program can take a few seconds to load it
  • Some fields are dependant upon the choices of other fields. eg. The "Dark description" will only be availabe to edit if "Can be dark" is checked
  • To include an image, you need to place it into the "imgs" folder in your game folder. Then select it at the "Image location" field
  • If you want the room to initially be unfound, ensure you uncheck (grey out) the "Is revealed" field
